¿Qué es Angular?

Angular es un framework web que empodera a los desarrolladores para construir aplicaciones rápidas y confiables. Es una plataforma de desarrollo construida sobre TypeScript, que incluye un framework basado en componentes para crear aplicaciones web escalables. Mantenido por un equipo dedicado en Google, ofrece herramientas, APIs y bibliotecas para simplificar el flujo de trabajo de desarrollo.

Concepto: Angular es un framework de frontend para SPAs (Single Page Applications).

¿Qué es? Una plataforma completa para apps web modernas.

¿Qué debo instalar?

Para comenzar con Angular 19, necesitas Node.js (versión 18 o superior) y el Angular CLI.

  1. Instala Node.js desde nodejs.org.
  2. Instala el CLI globalmente: npm install -g @angular/cli
npm install -g @angular/cli

Explicación: El CLI es la herramienta de línea de comandos para generar y gestionar proyectos Angular.

¿Cómo se crea un proyecto?

  1. Ejecuta: ng new mi-proyecto-angular
  2. Selecciona opciones (routing: yes, styles: CSS).
  3. Entra al directorio: cd mi-proyecto-angular
  4. Sirve la app: ng serve
ng new mi-proyecto-angular
cd mi-proyecto-angular
ng serve

Explicación detallada: ng new genera la estructura básica, instala dependencias y crea un app inicial. ng serve compila y sirve en localhost:4200.

La Estructura de un Proyecto Angular

El proyecto se organiza en un workspace con carpetas clave en src/.

  • app/: Componentes, módulos, servicios.
  • assets/: Imágenes, fuentes.
  • environments/: Configuraciones.
  • main.ts: Punto de entrada.
  • index.html: Plantilla base.

Concepto: Estructura modular para escalabilidad.

Explicación: El CLI crea un esqueleto listo para desarrollo, con archivos TypeScript, HTML y CSS.

Componentes en Angular y Cómo Llega el Contenido a la Pantalla

Los componentes son bloques de construcción que controlan una porción de la pantalla. El contenido llega vía detección de cambios: Angular actualiza la vista cuando el modelo cambia.

Concepto: Un componente encapsula template, lógica y estilos.

¿Qué es? Clase TS con decorator @Component.

import { Component } from '@angular/core';

@Component({
  selector: 'app-example',
  template: '
Hola
',
})
export class ExampleComponent {}

Explicación: El template se renderiza en el DOM cuando el selector se usa en otro template.

Crear el Primer Componente Personalizado en Angular

En nuestro proyecto de perfil de usuario, creamos un componente header.

  1. Ejecuta: ng generate component header
ng g c header

Explicación del código: Genera header.component.ts, .html, .css e .spec.ts. La clase define la lógica.

Configurar el Componente Personalizado en Angular

Declara el componente en app.module.ts (automático con CLI).

import { HeaderComponent } from './header/header.component';

@NgModule({
  declarations: [AppComponent, HeaderComponent],
  // ...
})

Explicación: Las declaraciones registran el componente para uso en el módulo.

Usar el Componente Personalizado en Angular

Inserta el selector en app.component.html.

<app-header></app-header>
<router-outlet></router-outlet>

Explicación detallada: Angular renderiza el template del header cuando encuentra <app-header>.

Dar Estilo al Componente Header y Agregar una Imagen en Angular

En header.component.css:

h1 { color: #667eea; }
img { width: 100px; border-radius: 50%; }

En header.component.html:

<h1>Perfil de Usuario</h1>
<img src="assets/user.jpg" alt="Usuario">

Explicación: Estilos encapsulados, imagen de assets/.

Preparar la Información del Usuario (para mostrar contenido dinámico) en Angular

Define interfaz en user.ts:

export interface User {
  name: string;
  age: number;
}

¿Qué es? TypeScript para tipado fuerte.

Almacenar Información en la Clase de un Componente en Angular

En user-profile.component.ts:

export class UserProfileComponent {
  user: User = { name: 'Juan', age: 30 };
}

Explicación: Propiedades de clase almacenan estado.

Mostrar Contenido Dinámico con Interpolación de Cadenas en Angular

En template:

<p>Hola, {{user.name}}! Tienes {{user.age}} años.</p>

Concepto: {{ }} inserta valores del componente.

Vinculación de Propiedades y Mostrar Valores Calculados en Angular

Propiedad: [src]="imageUrl"

Calculado: Usa getter.

<button [disabled]="!user.active">Activar</button>

Explicación: [] une propiedades del DOM a expresiones.

Vinculación de Atributos en Angular

<img [attr.alt]="'Foto de ' + user.name">

Explicación: attr. para atributos HTML dinámicos.

Usar Getters para Valores Calculados en Angular

get fullInfo() {
  return `${this.user.name} (${this.user.age} años)`;
}
<p>{{fullInfo}}</p>

Explicación: Getters computan en tiempo real.

Escuchar Eventos con Vinculación de Eventos en Angular

<button (click)="updateAge()">Cumpleaños</button>
updateAge() {
  this.user.age++;
}

Concepto: () une eventos a métodos.

Gestionar el Estado y Modificar Datos en Angular

Usa métodos para mutar estado, lifecycle hooks para timing.

ngOnInit() {
  // Inicializar estado
}

updateState() {
  this.user.name = 'Nuevo Nombre';
}

Explicación detallada: ngOnInit para init, ngOnChanges para inputs. Evita leaks en ngOnDestroy.
